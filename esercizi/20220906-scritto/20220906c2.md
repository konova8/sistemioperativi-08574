# Esercizio c.2
Usando i semafori implementare un servizio che preveda due funzioni:

`void sumstop(int v)`
`int sumgo(void)`

La funzione `sumstop` deve mettere il processo chiamante in attesa.
La funzione `sumgo` deve sbloccare tutti i processi messi in attesa con la `sumstop` e restituire la somma algebrica dei valori passati come parametro alla `sumstop` dai processi che sono stati sbloccati (zero se la `sumgo` viene richiamata quando non c'Ã¨ nessun processo bloccato)

---

```
Queue q[Semaphore] = new Queue();
Semaphore mutex = new Semaphore(1);
int sum = 0;

void sumstop(int v)
{
    mutex.P();  // Prendo la mutua esclusione
    Semaphore s = new Semaphore(0);
    q.enqueue(s);   // Metto il semaforo legato al processo in coda
    sum += v;
    mutex.V();  // Lascio mutex
    s.P();  // Blocco il processo
    free(s);
}

int sumgo()
{
    mutex.P();
    int localSum = sum;
    sum = 0;
    while (q.size() > 0)
    {
        s = q.dequeue();
        s.V();
    }
    mutex.V();
    return localSum;
}
```
